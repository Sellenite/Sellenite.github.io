JavaScript链式调用原理

发表于 2017-10-22

原理

链式调用的原理就是通过构造函数中的每个方法，在结尾都返回实例本身，这样就可以通过调用方法后继续通过实例来进行链式调用：

/* underscore实现方式，与lodash不同，lodash是惰性调用 */
var _ = function (array) {
    // 建立数组副本方法
    this._value = Array.prototype.slice.apply(array)
}

_.forEach = function (array, fn) {
    array.forEach(function (item, index, array) {
        fn.apply(item, [item, index, array])
    })
}
_.map = function (array, fn) {
    return array.map(function (item, index, array) {
        return fn.apply(item, [item, index, array])
    })
}
_.chain = function (array) {
    return new _(array)
}

// 链式调用遍历，实例化后除了chain每个方法都返回实例本身，故可以链式调用
for (var i in _) {
    if (i !== 'chain') {
        // 除去顶端chain，将所有方法赋值给prototype
        _.prototype[i] = (function (i) {
            // 闭包执行全局变量i
            return function (fn) {
                // 取出参数，其实就是fn，相当于以下写法
                // var args = []
                // args.push(fn)
                var args = Array.prototype.slice.apply(arguments)
                // 将value放在数组第一个，用于apply传值
                args.unshift(this._value)
                this._value = _[i].apply(this, args);
                return this;
            }
        })(i);
    }
}

_.prototype.value = function () {
    return this._value
}

var a = [1, 2, 3]

_.forEach(a, function (item) {
    console.log(item)
})

console.log(_.map(a, function (item) {
    return ++item
}))

_.chain(a).map(function (item, index) {
    return item * index
}).value()

input placeholder问题　

在chrome 模拟移动端调试时，显示的非常正常，但是在真机上，placeholder里面的内容明显靠上，非常的不美观

在国外网站，对这个属性的兼容性处理，那就是不要设计input的line-height或者设置line-height为normal即可，

试了一下，虽然在谷歌模拟调试里稍微偏上，但是在“真机上”正常垂直居中～

line－height

line-height经常用于文字居中，不同手机显示效果不一样。

解决方案：稍微大一点的高度，最好把line-height设置为高度+1px，两个平台显示都不会太‘奇怪’。

JS函数节流和函数去抖

发表于 2017-07-21

函数节流throttle.js

我们这里说的throttle就是函数节流的意思。再说的通俗一点就是函数调用的频度控制器，是连续执行时间间隔控制。主要应用的场景比如：

  • 鼠标移动，mousemove 事件

  • DOM 元素动态定位，window对象的resize和scroll 事件

函数去抖debounce.js

debounce和throttle很像，debounce是空闲时间必须大于或等于 一定值的时候，才会执行调用方法。debounce是空闲时间的间隔控制。比如我们做autocomplete，这时需要我们很好的控制输入文字时调用方法时间间隔。一般时第一个输入的字符马上开始调用，根据一定的时间间隔重复调用执行的方法。对于变态的输入，比如按住某一个建不放的时候特别有用。
debounce主要应用的场景比如：

  • 文本输入keydown 事件，keyup 事件，例如做autocomplete
