水平居中方法总结

发表于 2017-04-20

  • 若是行内元素, 给其父元素设置 text-align:center,即可实现行内元素水平居中.

  • 若是块级元素且定宽, 该元素设置 margin:0 auto 即可.

  • 若是 display:inline-block, 该元素的父元素设置 text-align:center 即可.

  • 若不论是块级还是行内，而且元素不定宽，可以给元素设置 display: table 然后设置 margin: 0 auto 即可.

单行文本
